## Description
A simple WPF desktop application that allows users to add, view, and delete contact information.

## Features
- Add contacts (Name, Phone, Email)
- Display contacts in a table (DataGrid)
- Delete selected contacts
- Input validation for phone numbers
- In-memory storage

## Tech Stack
- C#
- WPF (.NET)
- XAML

## Note
Contacts are stored in memory for simplicity and can be extended to file or database storage.
